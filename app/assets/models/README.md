# Модель распознавания лица (опционально)

**По умолчанию распознавание выполняется на сервере** (backend сам считает
эмбеддинг из загруженного фото — это надёжнее: клиенту не доверяем). Поэтому
для обычной работы **эта папка может быть пустой**, и мобильному приложению
модель не нужна.

## Когда всё же нужна модель на устройстве
Если хотите считать вектор лица прямо на телефоне (например, для офлайн-режима
или как дополнительный сигнал), положите сюда файл:

    mobilefacenet.tflite

- Формат MobileFaceNet / FaceNet, вход `112×112×3`, нормализация `[-1, 1]`.
- Приложение подхватит модель автоматически (`FaceService.modelReady`) и
  будет отправлять вектор вместе с фото. Если файла нет — просто отправляется фото.

Параметры входа/выхода при необходимости меняются в
`lib/services/face_service.dart`.
