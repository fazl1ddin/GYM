# Модель распознавания лица

Сюда нужно положить TFLite-модель эмбеддинга лица под именем:

    mobilefacenet.tflite

Приложение вычисляет по ней 128/192-мерный вектор лица на устройстве и
отправляет его на сервер для сравнения с эталоном (клиенту не доверяем —
итоговое решение принимает сервер).

## Где взять модель
- Подойдёт любая модель **MobileFaceNet / FaceNet** в формате `.tflite`
  с входом `112×112×3` (нормализация в диапазон `[-1, 1]`).
- Популярные открытые варианты — MobileFaceNet (~5 МБ). Скачайте `.tflite`
  и переименуйте в `mobilefacenet.tflite`.

## Если формат входа/выхода отличается
Поправьте параметры в `lib/services/face_service.dart`:
- `_embSize` — размер входа (по умолчанию 112);
- нормализацию пикселей (`(p.r - 127.5) / 128`);
- длина выходного вектора берётся автоматически из тензора модели.

> Без этого файла приложение запускается и показывает весь интерфейс,
> но при регистрации лица/отметке выдаст подсказку добавить модель.
