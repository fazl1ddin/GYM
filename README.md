# FaceClock

Мобильное приложение для учёта прихода и ухода с работы через камеру
с распознаванием лица и защитой от недобросовестных отметок.

## Статус
🔵 Этап проектирования — согласование ТЗ и выбор дизайна.

## Что уже есть
- **`docs/ТЗ.md`** — техническое задание. Ключевой раздел 6 — многоуровневая
  защита от «отметок за друга» (проверка живости, геозона, серверное время,
  привязка устройства, динамический QR/BLE, скоринг риска).
- **`designs/варианты-дизайна.html`** — три варианта визуального направления
  (открыть в браузере). Показаны экраны «Главная / статус смены» и
  «Сканирование лица» для каждого:
  - **A · Ясный день** — светлый корпоративный, для офиса/HR.
  - **B · Ночная смена** — тёмная тема, камера в центре, для 24/7 и IT.
  - **C · Просто и крупно** — крупные кнопки, для стройки/ритейла/цеха.

## Дальше
1. Выбрать вариант дизайна (A / B / C или гибрид).
2. Согласовать ТЗ (см. открытые вопросы, раздел 10).
3. Зафиксировать стек и приступить к MVP.
