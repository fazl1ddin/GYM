# FaceClock — Developer Handoff Spec

Учёт рабочего времени по распознаванию лица (Flutter, iOS). Источник дизайна — доски Claude Design (`FaceClock App.dc.html`, `FaceClock Admin & Auth.dc.html`); реализация — `app/lib`. Токены ниже совпадают с `app/lib/theme.dart`.

## Overview
Два контура по роли: **сотрудник** (отметка прихода/ухода по лицу, регистрация лица, история) и **администратор** (обзор-статистика, сотрудники, аномалии, табель, рабочие места с геозоной). Аутентификация — логин/пароль, сессия по cookie. Фон приложения светлый, единый акцент — синий.

**Типографика:** `.SF Pro Text` (системный iOS). Заголовок экрана 24/w800; секция 20/w800; число-метрика 30–34/w800; тело 15/w400–600; подпись 12–13/w600 `ink-soft`; чип 11–12/w700–800.

**Скругления:** карточка `18`, кнопка `15`, поле ввода `14`, плашка иконки `11–12`, чип/пилюля `999`.
**Тень карточки:** `0 12 22 rgba(32,58,90,0.07)` (SoftCard: `Color(0x14203A5A)`, blur 22, offset (0,12)).
**Сетка/отступы:** внешний паддинг экрана `18`; между карточками `8–12`; внутренний паддинг карточки `14–18`.

## Screens & States

### Login (`login_screen.dart`)
Логотип-плашка 66×66 (градиент accent→`#57C8FF`, br20, тень) слева, «FaceClock» 30/w800, подпись; карточка с полями Логин/Пароль (иконка + плейсхолдер), CTA «Войти», текст-кнопка «Настройка сервера».
- **Loading:** спиннер в кнопке, кнопка disabled.
- **Error:** текст `danger` под полями.
- **Config:** диалог ввода baseUrl (по умолчанию `--dart-define=FACECLOCK_BASE_URL`, иначе LAN).

### Employee Home (`home_screen.dart`)
Приветствие + аватар; баннер «Зарегистрируйте лицо» (если не зарегистрирован); баннер офлайн-очереди; карта статуса смены с большим таймером; CTA «Отметить приход/уход» (accent для «in», ink для «out»); последние отметки (StatusPill).

### Face Enrollment (`enroll_screen.dart`) — guided liveness
Камера с овалом + кольцо прогресса; чипы шагов **Центр → Поворот → Моргните** (done=success, active=accent). После прохождения живости — **ручная** кнопка «Снять и зарегистрировать» (не автоотправка). Снимок выпрямляется по EXIF; лицо валидирует сервер.
- **HR-режим:** admin регистрирует лицо сотрудника из карточки (тот же экран, `employeeId`).
- **Error:** snackbar, сброс к шагу «Центр», поток камеры перезапускается.

### Check-in (`checkin_screen.dart`)
3 чипа статуса сверху: **Геозона** (В зоне/Вне зоны/Подмена?/Поиск — клиент считает `distanceBetween`), **QR проходной** (Готово/Нужен скан/Не нужен), **Живость** (Проверка/Пройдена). Камера с liveness-челленджем. После живости — **ручная** кнопка «Отметить приход/уход». QR-гейт, если место требует QR.

### Admin Overview (`admin_home.dart` `_OverviewTab`)
Секции стат-карточек (grid 2 кол., aspect 1.32): **Сегодня** (Приходов/На смене/Опоздали/Отсутствуют/На проверке/Событий), **Проверки сегодня** (Подтверждено/Отклонено/Высокий риск/Офлайн), **Всего** (Сотрудников/Лицо зарег./Мест). Источник — `/api/admin/stats` (13 метрик).

### Employee Detail (`employee_detail_screen.dart`)
Профиль (аватар, имя, @логин, мини-метрики Приходов/Лицо/Устройство) + журнал отметок: тип/время/StatusPill, чипы **место · дистанция · QR · офлайн**, строка риска с флагами. Тап по карточке в списке сотрудников.

### Workplaces + Geofence (`workplaces_screen.dart`, `map_picker_screen.dart`)
Карточки мест: мини-карта, название, адрес, акцентная строка «радиус · координаты», QR-кнопка (soft) + удаление (soft danger). Тап по карточке → **редактирование** (bottom sheet, предзаполнено). Геозона — полноэкранная карта OSM: центральный пин, окружность-радиус в метрах, слайдер 50–1000м, «Мои координаты».

### Others
Anomalies (фото, риск-балл, флаги, Отклонить/Подтвердить), Timesheet (диапазон дат + CSV + таблица), QR-терминал (крупный QR + таймер 30с), QR-скан (камера + уголки-брекеты).

## Edge Cases
- **Пустые состояния:** `EmptyState` (сотрудники, места, табель, детали, аномалии).
- **Нет сети:** HTTP-таймаут 12с; bootstrap без сессии не ходит в сеть; офлайн-очередь отметок.
- **Длинный текст:** имена — `TextOverflow.ellipsis`; адреса переносятся.
- **Нет геозоны у места:** строка «координаты не заданы» (`warning`).
- **iOS ATS:** HTTP разрешён только для локальной сети (`NSAllowsLocalNetworking`); прод — HTTPS.

## Accessibility
- Хиты нижней навигации и кнопок ≥ 44pt.
- Тап вне поля прячет клавиатуру (`MaterialApp.builder`); контроллеры освобождаются (`dispose`).
- Контраст текста: `ink` на `panel`/`bg` — AA; проверить `ink-soft` на мелком тексте.
- **Плейсхолдеры:** камера/гео требуют `NSCameraUsageDescription`, `NSLocationWhenInUseUsageDescription`.
